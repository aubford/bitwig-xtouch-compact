MIXER
F1-F8 Mixer volume of 8 channel bank.
F9 Mixer volume currently selected track.
B1-B8 Mute of 8 mixer channels.
B9 + B1-B8 Solo of 8 mixer channels.
B9 + F9 Master volume.
K1-K8 Pan of 8 mixer channels.
INST DEFAULT
F1-F8 As per blue screen printing (if applicable).
F9 As per blue screen printing (if applicable).
K1-K8 As per blue screen printing (if applicable).
B1-B8 Switch devices 1-8 on/off in chain.
Shift + B1-B8 Select devices 1-8 in chain.
INST USER
Page Opens Bitwig Remote Controls Pane for current device
Shift + Page Toggles Bitwig Remote Controls Pane for current device Open/Close
Page + forward/Rewind Navigate/select Bitwig Remote Controls Pages
F1-F9 Assigned to Bitwig Remote Controls Page: Envelope.
K1-K8 Assigned to currently selected Bitwig Remote Controls Page.
B1-B8 Switch devices 1-8 on/off in chain.
Shift + B1-B8 Select devices 1-8 in chain.
TRANSPORT
« Transport rewind.
» Transport fast forward.
Stop Transport stop.
Play Transport play.
Rec Transport record.
Loop Transport toggle loop.

SHIFT FUNCTIONS
Shift+« Goto loop start.
Shift+» Set Loop Start to the current song position.
Shift+Stop Set Loop End to the current song position.
Shift+Play Undo.
Shift+Rec Click on/off.
Shift+Loop Overdub on/off.
CLIP TRIGGERING
Loop Trigger scene in selected row.
Mixer+« Previous row of clips.
Mixer+» Next row of clips.
Mixer+stop Stop clip.
Mixer+play Launch clip.
Mixer+rec Launcher overdub on/off.
CLIPS BUTTON
Clips + Pads 1-8 Selects clip banks 1-8 for a total of 64 clips for current track
Pads 1-8 If clip is empty, hitting a pad will start recording (red). If clip has content, it will play (green).
Shift + Pads 1-8 If clip is empty (off), hitting a pad will set a fixed length. If clip has content (yellow), it will be deleted.
SCENES BUTTON
Scenes + Pads 1-8 Selects scenes banks 1-8 for a total of 64 clips for current project.
Pads 1-8 Activate scenes in current bank. If Pad led is off, there is nothing to play.
Shift + Pads 1-8 Hit a pad to select 1 of 4 colors for the associated scene.
