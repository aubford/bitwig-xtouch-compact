BEHRINGER – BCR2000

Important note: For the script to work, you need to load the Bitwig Studio preset into the BCR2000 using the BC-Manager.
GLOBAL
EB1 Alt.
EB2 Toggle between Sends and Device Page.
EB3 Shift.
PE1-8 Macros of the first device on the selected track. Click on encoder in order to map a macro.
UB1 Previous Track Bank.
UB2 Next Track Bank.
UB3 Note Editor View.
UB4 Device View.
Shift + UB3 Automation Editor View.
Shift + UB4 Mixer View.
SENDS PAGE
SENDS
E17-24 3rd Send.
E9-16 2nd Send.
E1-8 1st Send.

B1-8

 Toggle Solo.
B9-16 Select track.
Alt + B1-8 Toggle Mute.
Alt + B9-16 Toggle Arm.
PB1&2 Previous/next Preset.
Shift (EB3) + preset buttons (PB1&2): Previous/next Preset Category.
DEVICE PAGE
DEVICE
E17-24 Parameters of cursor device.
E9-16 Envelopes of cursor device.
E1-8 Common parameters of cursor device.

B1-8

 Map modulation sources.
B9-16 Select track.
Alt + B1-8  
Alt + B9-16  
PB1&2 previous/next Preset.
Shift (EB3) + preset buttons (PB1&2): previous/next Preset Category.
