RELOOP – KEYPAD

Important note:

For the script to work, you need to set the KeyPad to Preset 2 by unplugging the USB cable and then holding Pad 2 while plugging the USB cable back in.

To set the controller to use external sync for the arpeggiator and run in sync with Bitwig Studio, unplug the USB cable, then hold down the ARP button while plugging the cable back in.
Now channel buttons 1 and 8 (S A1 + A8) should light up. Press channel button 8 (S A8) for external sync.
